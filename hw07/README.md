Требования к реализации:

0. Подключить spring-boot-starter-data-jpa +
1. Переписать все репозитории на Spring Data JPA репозитории +
2. Все тесты с прошлой работы должны проходить +
3. Все требования к прошлой работе, касающиеся работы со связями и транзакциями,
   должны быть также соблюдены (N+1, Lazy поля + equals/hashCode/toString и т.д.) +
4. Кастомные методы репозиториев (или с хитрым @Query) покрыть тестами, используя H2 +
5. @Transactional рекомендуется ставить на методы сервисов, а не репозиториев +
